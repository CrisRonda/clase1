# Javascript

`var, let, const`
`const`: constante no se puede cambiar su valor 
```js
const nombre = "Sandy"
const nombre= 1234
const nombre= [1,2,3,4]
```
**Excepci√≥n con los tipos array & objecto**

`let`: podemos cambiar cual quier valor
```js
let nombre = "Sandy"
nombre= 1234
nombre= [1,2,3,4]
```

`var`: funciona igual `let`, no utilizar var por el hoisting
